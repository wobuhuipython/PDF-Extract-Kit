# OCR 图片过滤功能

## 功能说明

新增智能 OCR 过滤功能，自动检测并过滤掉以下类型的无效图片：
- 空白图片（无文字）
- 装饰性图片（logo、图标等）
- 示意图/概念图（如"特许4S店"、"体验店"等场景展示图）
- 标签图（只有简短标签，无数据内容）

只保留真正有数据价值的图表，避免浪费 AI 分析资源。

## 安装依赖

```bash
pip install paddleocr paddlepaddle
```

如果有 GPU，可以安装 GPU 版本：
```bash
pip install paddleocr paddlepaddle-gpu
```

## 配置

在 `.env` 文件中添加以下配置：

```env
# OCR 过滤配置
ENABLE_OCR_FILTER=true          # 是否启用 OCR 过滤（默认: true）
MIN_TEXT_LENGTH=5               # 最小文字长度阈值（默认: 5）
MIN_NUMBER_COUNT=3              # 最小数字数量阈值（默认: 3）
OCR_STRICT_MODE=true            # 严格模式（默认: true，宁可错杀不可放过）
```

## 过滤模式

### 严格模式（推荐，默认开启）
**原则：宁可错杀，不可放过 - 只保留明确的数据图表**

图片必须同时满足以下条件才会保留：
1. 数字数量 >= `MIN_NUMBER_COUNT`（默认 3 个）
2. 不是装饰性图片（logo、标签等）
3. 不是实物照片（如"图19: 汽车变速箱智能装配"）
4. 不是功能说明图（如"Sale 整车销售"）
5. 不是流程图/概念图
6. 不是示意图/场景图
7. 包含数据关键词 OR 数字数量 >= `MIN_NUMBER_COUNT * 1.5`

### 宽松模式（设置 `OCR_STRICT_MODE=false`）
只过滤明显无效的图片，对边界情况更宽容。

## 过滤规则（严格模式）

图片会被过滤掉，如果满足以下任一条件：

1. **数字不足**：数字少于 `MIN_NUMBER_COUNT` 个（默认 3 个）
2. **装饰性图片**：只有单个单词或短语（如"Python"、"体验店"）
3. **实物照片**：以"图XX："开头的照片标题（如"图19: 汽车变速箱智能装配"）
4. **功能说明图**：包含"英文+中文"模式（如"Sale 整车销售"）
5. **流程图/功能图**：包含"功能"、"流程"、"装配"、"机器人"等关键词
6. **示意图/场景图**：包含"店"、"超市"、"园区"等场景词汇
7. **标签图**：文字分散分布
8. **数据不足**：无数据关键词且数字较少

只有明确包含数据的图表才会被保留。

## 工作流程

1. **提取图表**：从 PDF 中提取所有图表
2. **OCR 检测**：使用 PaddleOCR 检测每张图片中的文字
3. **智能分析**：
   - 统计文字数量
   - 统计数字数量
   - 检测是否包含数据关键词（如"数据"、"统计"、"%"等）
   - 检测是否是示意图关键词（如"店"、"园区"等）
   - 判断文字分布密度
4. **过滤判断**：综合以上因素判断是否保留
5. **AI 分析**：只对保留的图片进行 AI 分析

## 参数调优建议

### MIN_TEXT_LENGTH（最小文字长度）
- **默认值**: 5
- **建议范围**: 3-10
- **说明**: 太小可能保留无意义图片，太大可能过滤掉简洁的数据图表

### MIN_NUMBER_COUNT（最小数字数量）
- **默认值**: 3
- **建议范围**: 3-5
- **说明**: 
  - 设为 3：适合一般的数据图表（推荐）
  - 设为 4：更严格，只保留数据密集的图表
  - 设为 5：极严格，只保留数据非常丰富的图表
  - 不建议设为 2，会放过很多无效图片

### OCR_STRICT_MODE（严格模式）
- **默认值**: true
- **说明**: 
  - true：严格模式，宁可错杀不可放过（推荐）
  - false：宽松模式，对边界情况更宽容

## 性能优化

- 使用轻量级 OCR 模型，速度快
- 只检测文字，不进行完整识别
- 可以通过 `ENABLE_OCR_FILTER=false` 禁用此功能

## 注意事项

1. 如果未安装 PaddleOCR，过滤功能会自动禁用，所有图片都会被保留
2. 如果图片是纯图形数据图表（如热力图、地图），可能会被误过滤，建议降低 `MIN_NUMBER_COUNT`
3. 首次运行会下载 PaddleOCR 模型，需要一些时间
